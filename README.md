# Salamtak

Salamtak is a smart medication reminder system built using Flutter and Arduino.

## Features
- Voice reminders
- Easy mode for elderly users
- Bluetooth smart button (Arduino-based)
- Caregiver monitoring
- Refill alerts
- Arabic & English interface

## Structure
- app/ -> Flutter application
- hardware/ -> Arduino smart button
- docs/ -> Documentation
- assets/ -> Poster and screenshots

## Run Flutter App
cd app
flutter pub get
flutter run
